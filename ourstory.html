Visual Studio Code

O que é o Visual Studio Code? Ele é uma plataforma onde você irá escrever os códigos para criar sua página web.

O Visual Studio Code é um editor de código-fonte feito pela Microsoft e disponibilizado para o Windows, o Linux e o macOS. Com ele, o usuário pode escrever e editar código em várias linguagens de programação, inclusive HTML, mas o que fazer quando você precisa ver como está progredindo o código HTML no meio de seu trabalho? Uma extensão ou complemento que facilita a execução de HTML dentro do Visual Studio Code é a melhor opção. O Terminal também pode ser utilizado para rodar arquivos desse tipo. Leia o artigo para aprender a executar um HTML no Visual Studio Code.

Instalando o VS Code

Para baixar o VS Code, acesse o site oficial da ferramenta e clique em Download for Windows. Assim que terminar o download, clique no arquivo para realizar a instalação. Para prosseguir, você precisará aceitar os termos de uso. Em seguida, escolha a pasta do seu computador onde será instalado o VS Code.

Tutorial de como criar uma pasta:

Clique com o botão direito do mouse em uma área vazia de sua área de trabalho: -A área de trabalho do computador é a tela principal que você vê logo após ligar o computador e fazer seu login. - No local escolhido, clique com o botão direito do mouse em um espaço vazio da janela. Selecione "Novo" > "Pasta" - No menu que aparece, passe o cursor sobre "Novo" e depois clique em "Pasta". Nomeie a nova pasta: - A nova pasta aparecerá com o nome "Nova pasta" já selecionado. Basta digitar o nome que deseja dar à pasta e pressionar Enter.

Agora como você irá colocar sua pasta dentro do VS Code e criar seus códigos:

1. Abrir o VS Code - Abra o Visual Studio Code.

2. Selecionar a Pasta no VS Code - Com o VS Code aberto, clique em File (Arquivo) na barra superior e depois em Open Folder... (Abrir Pasta...). - Navegue até a pasta que você criou na área de trabalho. - Selecione a pasta e clique em Select Folder (Selecionar Pasta). Isso abrirá a pasta no VS Code.

3. Criar o Arquivo index.html - Com a pasta aberta no VS Code, clique no ícone de novo arquivo (ícone de folha com um sinal de +) no painel lateral esquerdo. - Nomeie o arquivo como index.html e pressione Enter.

4. Salvar Automaticamente - Para habilitar o salvamento automático, clique em File (Arquivo) na barra superior. - Selecione a opção Auto Save (Salvamento Automático). Isso garante que o arquivo seja salvo automaticamente sempre que você fizer alterações.

5. Entrar na Pasta e Acessar o Arquivo no Navegador (Windows) - Vá até a área de trabalho do seu computador. - Encontre a pasta que você criou e dê um duplo clique nela para abrir. - Dentro da pasta, localize o arquivo index.html e dê um duplo clique nele. - O arquivo será aberto no seu navegador padrão, permitindo que você visualize a parte visual do site.

Atalhos

CTRL + C = Copiar
CTRL + V = Colar
CTRL + A = Selecionar todo o texto de um documento
CTRL + X = Recortar um texto
CTRL + S = Salvar o arquivo
CTRL + Z = Desfaz a última ação
CTRL + D = Seleciona a próxima palavra ou caractere igual ao atual
CTRL + F = Encontrar
CTRL + SHIFT + N = Abre uma nova janela do VS Code.
CTRL + SHIFT + P = Abre a caixa de comando, onde é possível pesquisar por funcionalidades específicas.
CTRL + SHIFT + L = Seleciona todas as ocorrências da seleção atual.
CTRL + SHIFT + X = Abre a barra de extensões, onde é possível instalar e gerenciar extensões do VS Code.
SHIFT + ⬆️ = Seleciona a linha de cima
SHIFT + ⬇️ = Seleciona a linha de baixo

Como utilizar o terminal?

O terminal integrado do Visual Studio Code, o VS Code, possibilita executar diferentes terminais e códigos, neste artigo vamos abordar os primeiros passos para consumir essa excelente ferramenta do editor da Microsoft.

Para abrir um novo terminal, você pode utilizar o comando Ctrl + Shift + ' ou clicar na aba superior em Terminal e depois em New Terminal.

Digite cd e depois o caminho do arquivo HTML;, e aperte ↵ Enter.
Com isso, você será levado à localização do arquivo HTML; se estiver na pasta “Documentos”, por exemplo, digite
cd \Usuários\nomedeusuário\Documentos e aperte “Enter”.
- Se o HTML; estiver salvo em uma unidade diferente da qual o sistema operacional está instalado, será necessário mudar a letra dela no Terminal antes de acessar o caminho do HTML;. Basta digitar a letra (por exemplo, D:, se for a unidade “D:”) e pressionar “Enter”.

- Caso não saiba exatamente a localização do HTML;, clique com o botão direito na guia dele, na parte superior da tela, e selecione “Copiar caminho”. Digite cd no Terminal e cole o caminho logo depois. Apague o nome do arquivo no fim do caminho e tecle “Enter”.

- Se alguma pasta do caminho até o diretório do HTML; apresentar um espaço no nome, o Terminal não conseguirá chegar até o destino. Use o Finder ou o Explorador de Arquivos (Windows) para acessar pastas com espaço no nome e então renomeie-as, removendo os espaços (por exemplo: a “Arquivos HTML” pode ser renomeada como “Arquivos_HTML;”).

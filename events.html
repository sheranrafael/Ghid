GitHub

O que é o GitHub? É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.

O GitHub é uma plataforma de desenvolvimento colaborativo que aloja projetos na nuvem utilizando o sistema de controle de versões chamado Git. A plataforma ajuda os desenvolvedores a armazenar e administrar o código e faz o registro de mudanças. Geralmente o código é aberto, o que permite realizar projetos compartilhados e manter o acompanhamento detalhado de seu progresso. A plataforma GitHub também funciona como rede social, conectando os desenvolvedores com os usuários. Como usuário, você pode descarregar programas ou aplicativos, e da mesma maneira, pode colaborar com seu desenvolvimento oferecendo melhorias e discutindo as questões que interessam nos fóruns temáticos.

Criando uma conta no GitHub

Para criar uma conta, primeiro você precisa entrar no site do GitHub. Já no site, no campo em branco no lado esquerdo da página, você deve digitar seu email para iniciar o cadastro. Em seguida, clicamos no botão verde escrito Sign up for GitHub No próximo passo, você deve confirmar este email, apertando em continue

Feito isso, precisamos criar uma senha que tenha entre 8 a 15 caracteres, incluindo números e letras minúsculas e um caractere especial, por exemplo, @ ! # $.
Nesse momento é importante que apareça a mensagem “Password is strong”, que indica que a sua senha é forte. Quando essa mensagem aparecer, aperte continue.

Agora você precisa adicionar um username (ou "nome do usuário", em português).

Finalizada essa parte, você vai precisar acessar seu email e confirmar o cadastro. Para fazer isso vá na sua caixa de email, o mesmo que foi utilizado no cadastro, e nela vai aparecer, na caixa de entrada, o seguinte email: “[GitHub] Please verify your email address”, em português: (“[GitHub] Por favor, verifique seu endereço de email”). Aqui você só precisa confirmar a verificação do email clicando no botão azul escrito “Verify email address" que significa “Verificar endereço de email”.

Pronto! Agora nós temos nossa conta no GitHub!

Repositórios

Existem vários lugares que servem como locais para depositar nossos arquivos e nossas páginas. Chamamos esses lugares de servidores.
Então, usaremos um lugar que servirá para publicar nosso código fonte, que são esses arquivos que conseguimos ver, compartilhar, armazenar, como também teremos acesso através do navegador: o GitHub. Estamos no GitHub, com nosso usuário logado. Vamos clicar no botão que tem o ícone de mais (+), porque queremos criar um lugar onde colocaremos nossos arquivos.
Esse lugar é um repositório. Então, novo repositório. O GitHub em inglês estará escrito New Repository. É nele que vamos clicar.

Depois, abaixo, tem a opção de criarmos esse código de forma pública, para que outras pessoas possam acessar tanto o código quanto a página, que é o que queremos. Em seguida, vamos apertar o botão Create Repository, localizado no canto inferior da página. Ele criará o repositório para nós e mostrará uma página que informa que o repositório ainda não tem nenhum arquivo.
Vamos fazer o upload dos nossos arquivos. Para isso, clicaremos em upload an existing file (subir um arquivo que já existe).
Feito isso, teremos adicionado os arquivos ao repositório. Agora, precisamos salvá-los. Então, no campo chamado "Commit changes", vamos adicionar a descrição: "Primeira versão do código". Agora, podemos apertar o botão de efetuar as mudanças, o Commit changes.

Deploy nos projetos

Para utilizar o GitHub Pages em nossos projetos, precisamos seguir alguns passos.
O ponto de partida é o nosso repositório, nele você vai acessar a opção Settings
Após isso, será exibida na página a seção General, como indica na imagem abaixo. No lado esquerdo da página, temos o campo Code and automation e na última opção desse campo está disponível a opção Pages.
Assim que identificar essa opção clique nela. Feito isso, conseguimos acessar algumas informações sobre o GitHub Pages.
Mais abaixo da tela, temos o campo Branch com as opções None e Save. Clique em None e altere para Main. Feito isso, confirme clicando em Save.
Caso siga todos esses passos, a mensagem GitHub Pages source saved (em português, GitHub Pages foi salvo) será exibida na parte superior da tela.
